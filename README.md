# Library Management System in C++

## 📖 Overview
The Library Management System is a C++ console application that demonstrates Object-Oriented Programming (OOP) and File Handling concepts. It enables users to manage books, search by title or author, issue and return books, and store records permanently using a binary file.

## 🚀 Features
- Add new books
- View all books
- Search books by title
- Search books by author
- Issue books
- Return books
- Persistent data storage using file handling
- Menu-driven interface

## 🛠️ Technologies Used
- C++
- Object-Oriented Programming (Classes & Objects)
- File Handling
- Binary Files

## 📂 Project Structure
```
LibraryManagementSystem.cpp
library.dat (created automatically)
README.md
```

## ▶️ How to Run
1. Compile the program:
   ```
   g++ LibraryManagementSystem.cpp -o library
   ```
2. Run the executable:
   ```
   ./library
   ```
   *(Windows users can run `library.exe`.)*

## 📋 Menu Options
1. Add Book
2. View All Books
3. Search by Title
4. Search by Author
5. Issue Book
6. Return Book
7. Exit
